1 · Prep

Physical size drives the stereo solve, dust scale and DOF. Web images carry no real size — give a width or pick a preset, or the item is assumed 300 ppi and flagged. Transparent PNG/WebP clipouts use their own alpha as the matte.

2 · Look

Set resolution before rendering — the halftone ruling is computed from it.

1.0% parallax for anaglyph. Interaxial is solved per item — never assumed.
